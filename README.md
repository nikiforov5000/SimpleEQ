<h3>VST3 Audio plugin</h3>

* **Low-cut / high-cut filter** 
	* frequency control 
	* slope control: 12, 24, 36, and 48 dB per octave 
*  **Bell filter** 
	* frequency control 
	* band Quality control: 0.1 - 30 
	* gain control: -48 - +48 dB
